# Ruby Automation
Automation framework written in Ruby and Cucumber Gherkin. Aims to follow Cheezy's PageObject implementation pattern.

## Requirements
* Ruby v2.7.0
* Google Chrome v83.0.4103.39
* ChromeDriver 83.0.4103.39
